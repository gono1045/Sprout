### 1. ヘッダーレイアウトの修正（全シート共通）

**問題**  
各シートのヘッダー部分がセル幅に依存しているため、シートによってはレイアウトが崩れている。

**修正方針**  
- 全シートのカラム幅をデフォルト（100px）にリセットする  
- 幅が必要な箇所はセルの結合で対応する（セル幅による調整は禁止）  
- 全シートで統一したヘッダーレイアウトを適用する

**対象シート**  
表紙、改訂履歴、画面遷移図、画面レイアウト、DB操作定義、画面項目定義、項目操作可否、画面機能一覧、画面機能仕様（全9シート）

### 2. 改訂履歴シートのレイアウト修正

**問題**  
- 列A：内容なし・不要な空白列
- 列E：異常に幅が広く、大きな空白ができている  
- 列G〜L：ほぼ空でページ右端まで無駄な空白がある  
- テーブル（行5以降）が列Fまでしか使われておらず、G〜Lが空のまま囲まれている

**修正方針**  
ページ横幅いっぱいまで使うよう再レイアウトする。以下を目安とした構成にする：

```
行1（ヘッダー情報）:
  A1:B1  → "システム名"
  C1:E1  → "Sprout"
  F1:G1  → "画面名"
  H1:J1  → "ホーム画面"
  K1:L1  → "ドキュメント名"
  M1:N1  → "基本仕様書"

行2（ヘッダー情報）:
  A2:B2  → "作成日"
  C2:E2  → "2026/05/03"
  F2:G2  → "更新日"
  H2:J2  → "2026/05/03"
  K2:L2  → "版数"
  M2:N2  → "1.0"

行4（タイトル）:
  A4:N4  → "改訂履歴"（中央揃え、結合）

行5（テーブルヘッダー）:
  A5:B5  → "版数"
  C5:D5  → "更新日"
  E5:F5  → "更新者"
  G5:K5  → "変更内容"
  L5:N5  → "備考"

行6以降（テーブルデータ）: 上記に合わせて入力
```

### 3. A4ページ枠の設定（全シート）

**問題**  
画面レイアウトシートのページ枠線がA4サイズになっていない（他シートも同様）。

**修正方針**  
- 全シートの印刷設定をA4縦（210mm × 297mm）に統一する  
- 印刷範囲をコンテンツに合わせて設定する  
- ページ区切り線がA4サイズで表示されるよう調整する

---

### 4. 画面遷移図の文字引き伸ばし修正

**問題**  
画面遷移図内の図形（モーダル表示ラベル等）のテキストが横方向に引き伸ばされて歪んで見える。

**修正方針**  
- 該当の図形テキストのフォントサイズ・スタイルを確認・修正する  
- 図形の縦横比を正しいアスペクト比に戻す  
- テキストが折り返しまたは適切にフィットするよう調整する

### 5. ワイヤーフレームの作成・更新（画面レイアウトシート）

**問題**  
現在のワイヤーフレームが実際のアプリ画面と一致しておらず、デザインも古い。

**修正方針**  
以下の2種類のワイヤーフレームを画面レイアウトシートに掲載する：

#### アプリ概要（Sprout）
- タスク管理 + ゲーミフィケーション要素を持つアプリ
- 背景：ダークフォレスト風
- メイン画面要素：タスクリストテーブル、植物レベルアップカード

#### デフォルトUI（ライトモード）ワイヤーフレーム
```
┌─────────────────────────┐
│ ナビバー（Sprout ロゴ）       │
├─────────────────────────┤
│ 植物カード（経験値・レベル）    │
├─────────────────────────┤
│ [タスク追加ボタン]          │
├─────────────────────────┤
│ タスクリスト               │
│  ヘッダー行                │
│  行① チェックボックス｜タスク名│
│  行② チェックボックス｜タスク名│
│  行③ チェックボックス｜タスク名│
├─────────────────────────┤
│ 植物リスト（サムネイル一覧）   │
└─────────────────────────┘
カラー: 白背景、#2D6A4F（グリーン）アクセント
```

#### ダークモードUIワイヤーフレーム
- 上記と同じ構成・要素
- 背景色：ダーク（#1A1A2E 系）
- テキスト：白系
- アクセント：#52B788（ライトグリーン）
- カード背景：#2D2D3D 系

**補足**  
- シンプルなデザインで可（細かい作り込みは不要）  
- 両方を横並びまたは縦並びで1シート内に配置  
- Sprout のブランドカラー参考：`#2D6A4F`, `#B7E4C7`, `#52B788`, `#3B82F6`

## 技術的コンテキスト（作業上の注意）

- スプレッドシートはプライベートファイルのため、OAuth認証が必要
- Apps Script（script.google.com）へのJavaScriptアクセスはブロックされる場合がある
- Google Sheets API（sheets.googleapis.com）は別ドメインのためセッションCookieでは認証不可
- **推奨アプローチ**：Google Sheets の UI（ブラウザ操作）または Apps Script エディタから直接スクリプトを実行する
