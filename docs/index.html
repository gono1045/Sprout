タスクを育てる、
管理Webアプリ

Sprout は Spring Boot + Thymeleaf で構築したタスク管理アプリです。個人学習の集大成として、セキュリティ・アーキテクチャ・DBアクセスなど実務に近い技術を組み合わせて開発しました。

Tech Stack
Java 17 + Spring Boot 3.5
バックエンド基盤、MVC構成
Spring Security
認証・認可、カスタムUserDetails
JPA + MyBatis
ORM併用、用途に応じて使い分け
PostgreSQL
リレーショナルDB、JPA自動スキーマ
Thymeleaf
サーバーサイドレンダリング
Tailwind CSS 4.x
ユーティリティファーストCSS
Features

主な機能

ユーザー認証・登録
Spring Security によるセキュアなログイン。パスワードはBCryptでハッシュ化して保存。
タスク管理（CRUD）
タスクの作成・編集・削除・完了切り替え。モーダルUIでスムーズな操作感を実現。
タグ管理
タスクにタグを付与して分類。カラー付きタグで視覚的な整理が可能。
アクセス制御
他ユーザーのタスクへのアクセスを制御する AccessControlService を実装。
What I Learned

この開発で学んだこと

  • Spring Security のカスタム認証フローと UserDetailsService 実装
  • JPA と MyBatis の使い分け（基本CRUDはJPA、複雑なJOINはMyBatis）
  • レイヤードアーキテクチャ（Controller → Service Interface + Impl → DAO）
  • Thymeleaf テンプレートエンジンでのサーバーサイドレンダリング設計
  • Spring Validation によるフォームバリデーションとエラーハンドリング
  • 環境別の application.yml 管理と機密情報の .gitignore 除外
  • Tailwind CSS 4.x を npm ビルドで組み込むフロントエンド構成
  • Docker を使ったコンテナ化とデプロイ準備
  • グローバル例外ハンドラー（GlobalExceptionHandler）による統一エラー処理
  • カスタム Enum（SproutTagColor）を活用したドメインモデル設計
